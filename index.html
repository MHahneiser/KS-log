Synchronisierung (iPhone ↔ iPad)

Gleicht beide Geräte über einen privaten, Ende‑zu‑Ende‑verschlüsselten GitHub‑Gist ab. Im Gist liegen nur verschlüsselte Daten. Gib auf beiden Geräten dasselbe Token und dasselbe Sync‑Passwort ein.

Sync aus
Token erstellen: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → „Generate new token", nur das Häkchen bei gist setzen. Das Token wird ausschließlich lokal gespeichert. Wichtig: Verlierst du das Sync‑Passwort, sind die abgeglichenen Daten nicht mehr entschlüsselbar.

Daten & Sicherung

Alle Daten liegen ausschließlich auf diesem Gerät. Sichere regelmäßig per Backup‑Datei (JSON). Diese lässt sich wieder einspielen und z. B. in iCloud Drive / OneDrive ablegen.
